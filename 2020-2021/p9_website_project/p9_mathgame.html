Landing Page About Hobbies Math Minigame

How this works:
This is loosely based on a game from elementary school. You have 90 seconds to answer as many of the following math questions as you can. All answers will be integers. Represent negative numbers with the "-" sign. Sample answers include 8, -18, and 193. Pressing the enter key (more specifically, releasing the enter key) moves you on to the next question, which can be used to skip the current question. You get 1 point per correct question, and no penalty for skipping or answering incorrectly. The highscore does not persist after reloading the page.
